# DecibelimetroJavaClone

Projeto Android nativo em Java inspirado no layout do React/Figma.

## Requisitos
- Android Studio recente
- SDK 34
- JDK 17

## Como abrir
1. Extraia o zip
2. Abra a pasta no Android Studio
3. Aguarde o Gradle sincronizar
4. Rode em um emulador ou celular Android

## Observação
A leitura de dB é aproximada, baseada na amplitude do microfone.
